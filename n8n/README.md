# n8n

## Installation

Install **n8n** via npm

```bash
npm i -g n8n
```

## start

Run below command in **Command Prompt**

```bash
set NODES_EXCLUDE="[]" && set NODE_FUNCTION_ALLOW_EXTERNAL=uuid && set N8N_RESTRICT_FILE_ACCESS_TO="" && n8n
```

## Import workflow

Import **AI REST APIs Testing Workflow.json** file to your n8n workflows
